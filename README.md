# 项目简介 java493

## 项目概述

本项目是一款基于SSM+Vue的课程网络学习平台管理系统。采用JavaEE技术，实现了前后端分离，前端使用Vue.js框架，后端采用Spring + Spring MVC + MyBatis架构。系统遵循MVC三层体系结构，确保了良好的代码分层和可维护性。

## 技术栈

- **前端**: Vue.js
- **后端**: Spring, Spring MVC, MyBatis
- **构建工具**: Maven
- **数据库**: MySQL

## 功能模块

- 课程管理
- 用户管理
- 学习进度跟踪
- 成绩管理

## 系统角色

- 管理员
- 教师
- 学生

## 运行环境

- **操作系统**: Windows/Linux/MacOS
- **数据库**: MySQL 5.7+
- **JDK**: JDK 1.8+
- **Web 服务器**: Tomcat 8.5+

## 部署步骤

1. 准备好运行环境，包括JDK、MySQL和Tomcat。
2. 创建数据库，导入初始数据。
3. 部署后端代码到Tomcat。
4. 前端代码通过Vue.js编译后，部署到Web服务器。
